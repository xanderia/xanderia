# Xanderia Monorepo

## Structure

| Folder | Description |
| :----- | :---------- |
| `/`    | Configuration Files |
| `/xata` | XATA library, used by others projects |
| `/functions` | Firebase Functions for all server-side services |
| `/*` | Specific app, e.g. `/chat`, `/calendar`, `tasks` |
